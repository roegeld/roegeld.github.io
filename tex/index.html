Modules LaTeX des écoles doctorales
IAEM et SLTC

Cette page est provisoire et donne des informations sur le programme des modules LaTeX des écoles doctorales IAEM et SLTC. Les formations sont ouvertes aux étudiants d'autres écoles doctorales.

École doctorale IAEM :

  • Module LaTeX introductif, décembre 2025, IUT Charlemagne, Nancy (module de 10h en cinq séances)

    Ce module propose un tour d’horizon des outils liés à TEX pour la mise en page de document. La formation commencera à partir de zéro. Parmi les sujets abordés, il y aura notamment (un peu de) :

    • Structure de TEX : différence entre TeX et LaTeX, un peu d’historique, évolution future, ...
    • Recherche d’information, accession à l’autonomie TeX.
    • Installation de quelques packages.
    • Notions de codages en entrée et en sortie.
    • Polices.
    • Mise en page de mathématiques.
    • Insertion de figures déjà créées par ailleurs.
    • Dessin technique avec des outils comme metapost et TikZ.
    • Tableaux.
    • Bibliographie.
    • Index.
    • Typographie française, anglaise et style.
    • Création d’une classe LaTeX et plus généralement d’ensembles de commandes pour remplir un but particulier.
    • Présentations.

    En outre, il sera possible d’aborder d’autres questions en fonction des demandes et si le temps le permet.

    Il est souhaitable que les étudiants inscrits aient déjà une version de LaTeX installée. Pour Windows, une installation convenable est MikTeX, complétée par une interface comme TeXmaker, WinEdt, TeXworks, TeXstudio, etc. (une seule suffit). Des distributions LaTeX existent aussi pour Apple et linux. Il est aussi possible d’utiliser Overleaf qui propose une version gratuite en ligne, donc sans nécessité d’installer une version de LaTeX. Cela peut être une solution pour commencer.

École doctorale SLTC :

  • Module LaTeX général, janvier 2026, IUT Charlemagne, Nancy (module de 8h en quatre séances)

    Ce module propose un tour d’horizon des outils liés à TeX pour la mise en page de documents. Il pourra être complété par le module plus spécialisé sur le sujet du graphisme et des illustrations dans les documents.

    Le contenu est le même que celui du module introductif IAEM, si ce n'est que le sujet des mathématiques ne sera pas abordé, ou simplement effleuré. Ceux qui souhaiteraient en savoir plus sur ce sujet devraient s'inscrire au module de l'école IAEM. La question de la réalisation de figures, notamment techniques, sera développée dans le module spécialisé qui suivra celui-ci.

    En outre, il sera possible d’aborder d’autres questions en fonction des demandes et si le temps le permet.

    Il est souhaitable que les étudiants inscrits aient déjà une version de LaTeX installée. Pour Windows, une installation convenable est MikTeX, complétée par une interface comme TeXmaker, WinEdt, TeXworks, TeXstudio, etc. (une seule suffit). Des distributions LaTeX existent aussi pour Apple et linux. Il est aussi possible d’utiliser Overleaf qui propose une version gratuite en ligne, donc sans nécessité d’installer une version de LaTeX. Cela peut être une solution pour commencer.

  • Module LaTeX spécialisé graphisme, janvier 2026, IUT Charlemagne, Nancy (module de 10h en cinq séances) (attention : ce module nécessite d'avoir déjà de bonnes connaissances de LaTeX).

    Ce module s'adresse à des étudiants ayant déjà une bonne base de l’utilisation de LaTeX. Il propose un complément spécialisé sur la réalisation de dessins scientifiques et techniques avec la bibliothèque TikZ. TikZ est une bibliothèque de fonctionnalités qui facilitent grandement la réalisation de dessins complexes, mais sous forme de code LaTeX.

    Le programme est en gros celui de la page tutorial, à savoir :

    • exemple de base ;
    • coordonnées et grilles ;
    • tracé de segments, lignes polygonales ;
    • tracé de courbes ;
    • objets particuliers : cercles, rectangles, etc. ;
    • styles des lignes ;
    • remplissage, dégradés ;
    • intersections ;
    • fonctions, boucles ;
    • insertion de texte ;
    • utilisation de certaines bibliothèques TikZ ;
    • interfaces pour la réalisation de figures TikZ ;
    • etc.

    En outre, il sera possible d’aborder d’autres questions en fonction des demandes et si le temps le permet.

    Il est souhaitable que les étudiants inscrits aient déjà une version de LaTeX installée. Pour Windows, une installation convenable est MikTeX, complétée par une interface comme TeXmaker, WinEdt, TeXworks, TeXstudio, etc. (une seule suffit). Des distributions LaTeX existent aussi pour Apple et linux. Il est aussi possible d’utiliser Overleaf qui propose une version gratuite en ligne, donc sans nécessité d’installer une version de LaTeX. Cela peut être une solution pour commencer.

    Il y aura à l’issue de ce module un petit devoir à faire, sous forme d’une figure à refaire en TikZ.
